Sistema de horarios con Mapa de salones para Los alumnos de la Universidad Gran Mariscal De Ayacucho.

Este proyecto consiste en una pagina Web en la cual los alumnos puedan visualizar sus horas de clase a través de un horario y la ubicación de donde tendrán clase a través de un mapa de la universidad.

Esta pagina fue creada porque el horario ofrecido por la universidad no es claro en el sentido de que tienes que pasar el mouse por encima de las materias para ver su nombre, ademas de que he notado que mis compañeros no recuerdan donde están algunos salones.

El proyecto cuenta con dos roles: Usuario y Administrador.

El usuario puede seleccionar las materias que ve, visualizar su horario, visualizar mapa del piso y interactuar con el mapa de piso al hacer click en un salón le muestra las materias que vera en este junto con el día y hora, ademas que en el horario esta el nombre del salón.


El administrador puede Gestionar(Crear, editar, eliminar): Materias, secciones, carreras, horario de sección.

Pythrelius Versión programa de escritorio Editor Python con gestor de archivos.

La aplicacion cuenta con dos roles uno de usuario y otro administrador:

Usuario: El usuario puede crear proyectos y editar codigo de estos y subir archivos a este proyecto ademasde editar,clonar y renombrar proyectos y archivos.

Administrador: Puede controlar a los usuarios y editar ciertos parametros de estos ademas de que puede vizualizar los archivos de el proyecto del usuario para ver que no lo esten mal usando la BD.

Listado de fechas de reloot para el servidor Latin Force Vanilla.

Este proyecto consiste en una pagina web hecha para calcular la fecha y hora de cada respawn del servidor latin force vanilla.

Debido a que el la reaparición de objetos dentro del juego es cada 36 horas es difícil saber cuando sera el próximo respawn así que hice una pagina usando la fecha del primer respawn para que calcule las fechas y horas de los próximos.

Juego de carreras Career Renais.

Este proyecto consiste en un juego de carreras hecho en godot usando el lenguaje de propio motor GodotScript, este proyecto fue hecho para la materia Desarrollo de software.Aun que originalmente fue hecho para pc tambien hice un port para android.

El juego cuenta con 2 enemigos que serian el niño mimado y la religiosa, el jugador es un militar, todos estos personajes ficticios compiten en una carrera para obtener un nuevo cuerpo robotizado porque cada uno tiene sus cuerpos en mal estado.

Los enemigos son capaces de usar los objetos y de esquivar los obstáculos.

El juego también esta disponible para pc la única diferencia es que no hay controles táctiles usar teclas.

Nota: Yo hice el back-end del proyecto y el modelado 3D(Casino(atajo), Arboles, Bancos(parque), también hice algunos sprite: humo del auto,traga perlas, césped del parque y hojas de los arboles. el resto del arte fue realizado por mis compañeros de grupo.

Objetos:
Pistola: dispara una bala que al impactar contra jugador o enemigo lo congela
Medicamentos: Se puede usar 3 veces y da un empujón al jugador
Chicle: Te vuelve invulnerable temporalmente a las balas.

En la pista hay cajas de objetos que darán objetos al jugador y a los enemigos, el jugador para seleccionar deberá oprimir la combinacion de teclas(z-x-c) del objeto que quiere usar:
